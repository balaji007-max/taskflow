# TaskFlow – Team Task Manager

Full-stack web app with Node.js + Express + MySQL + JWT Auth + Role-Based Access Control.

## Quick Start

1. **Configure `.env`** – set your MySQL credentials  
2. **Init database** – `node server/database/setup.js`  
3. **Start server** – `npm start`  
4. **Open** – http://localhost:5000

## Deploy to Railway

See full guide in the deployment walkthrough artifact.

## Tech Stack
- **Backend**: Node.js, Express, MySQL2, JWT, bcrypt
- **Frontend**: Vanilla HTML/CSS/JS (dark glassmorphism UI)
- **Database**: MySQL with proper foreign keys & indexes
- **Auth**: JWT Bearer tokens, bcrypt password hashing
- **RBAC**: Global Admin / Project Admin / Member roles
